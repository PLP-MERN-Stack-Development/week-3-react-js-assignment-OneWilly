# React Task Manager

## Features
- Task management with local storage
- API data fetching with search
- Dark/light mode toggle
- Responsive design

## Setup
1. Clone repository: \`git clone <repo-url>\`
2. Install dependencies: \`npm install\`
3. Start dev server: \`npm run dev\`
